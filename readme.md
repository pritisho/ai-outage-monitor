# AI Outage Monitor

A  reliability dashboard that monitors AI providers like OpenAI and Claude.

## Features

- Live provider status
- Last 5 outages from official APIs
- Availability metrics
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- Requests
- Pandas

## Run Locally

python fetch_status.py  
python collect_incidents.py  
python -m streamlit run app.py

## Why This Project Matters

- API integration  
- Reliability monitoring  
- Incident tracking
